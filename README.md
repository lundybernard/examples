# Code Examples

[python error passing](./python_error_passing/): Unit tests 
for a Python function that returns errors instead of raising exceptions.


## Testing Examples
The code in these examples can be imported without side-effects,
and you can run the included unit tests to verify the code is correct.

### Using Pytest:
`pytest {example_directory}`

### Using Unittest (stdlib):
`python -m unittest discover -s {example_directory}`
